# Finance AI Radar

A curated landing page and searchable source library for applied AI in finance.

## Run

```bash
npm install --include=dev
npm run dev
```

Open http://localhost:3000.

## Verify

```bash
npm test
npm run typecheck
npm run build
```

## Editorial rule

Every item needs a source link, finance use case, why-it-matters rationale, caveat, score, review status, and license note. Broker research and expert-call material must stay rights-safe: metadata and original commentary only unless explicit rights allow more.
